# Website Formulir Pendaftaran Keanggotaan Klub

Website sederhana untuk formulir pendaftaran keanggotaan klub, dibuat menggunakan HTML, CSS, dan JavaScript murni (tanpa framework).

## Deskripsi

Website ini menyediakan formulir interaktif untuk pendaftaran keanggotaan klub. Formulir mencakup field untuk nama, email, nomor telepon, alamat, tanggal lahir, serta persetujuan aturan keanggotaan dan kebijakan privasi. Validasi dilakukan di sisi frontend untuk memastikan semua field diisi dengan benar sebelum pengiriman.

## Fitur

- **Formulir Lengkap**: Input untuk nama depan/belakang, email, telepon, alamat lengkap, dan tanggal lahir.
- **Validasi Real-time**: Feedback langsung saat mengisi field dan checkbox.
- **Autofill Support**: Mendukung autofill browser untuk pengalaman pengguna yang lebih baik.
- **Responsif**: Desain yang dapat menyesuaikan dengan berbagai ukuran layar.
- **Validasi Frontend**: Pengecekan field kosong dan persetujuan checkbox sebelum submit.

## Teknologi yang Digunakan

- **HTML5**: Struktur halaman web.
- **CSS3**: Styling dan layout responsif.
- **JavaScript (ES6+)**: Logika validasi dan interaktivitas.

## Cara Menjalankan

1. **Clone atau Download**: Unduh file proyek ke komputer Anda.
2. **Buka di Browser**: Buka file `index.html` langsung di browser web (Chrome, Firefox, Edge, dll.).
   - Tidak memerlukan server lokal karena ini adalah website statis.
3. **Isi Formulir**: Lengkapi semua field dan centang checkbox persetujuan.
4. **Kirim**: Klik tombol "Kirim Pengajuan" untuk validasi dan konfirmasi.

## Struktur Proyek
